# query word plugin for anki


![](demo.gif)

## 安装
- 下载wordquery.py和mdict文件夹到anki插件目录
- 或者，anki中使用代码775418273安装（不能保证及时更新）

## 使用
- 本插件需要模板支持，在anki中导入templates文件下的apkg文件
- “工具”菜单-->"word query"，设置词典文件的路径
- 添加新的单词时，点击Query按钮，从词典中读取释义。
